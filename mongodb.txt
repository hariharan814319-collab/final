# PHOENIX Hospital Management System – MongoDB Atlas Integration & Backend Debugging Prompt

## Project Overview

I have built a complete MERN Hospital Management System called **PHOENIX**.

The backend is built with:

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Nodemailer
* Multer
* REST APIs

The frontend is HTML, CSS and JavaScript.

The backend works correctly with my local MongoDB setup, but I am now migrating it to MongoDB Atlas for deployment.
